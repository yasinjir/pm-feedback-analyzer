# SignalForge Module Architecture

## 1. SignalForge Collect
Purpose: gather feedback and product inputs from different sources.

### Possible inputs
- forms
- surveys
- public links
- manual text input
- table import
- CSV import

## 2. SignalForge Insights
Purpose: turn raw feedback into structured product insight.

### Core outputs
- themes
- pain points
- feature requests
- opportunity signals
- summary reports
- priority cues

## 3. SignalForge PRD Studio
Purpose: convert insight into a standardized product requirements document.

### Core outputs
- problem statement
- goals
- users
- scope
- risks
- success metrics
- formatted PRD draft

## 4. SignalForge Tasks
Purpose: convert PRD output into execution-ready work.

### Core outputs
- task breakdown
- user stories
- acceptance criteria
- delivery-ready task lists
