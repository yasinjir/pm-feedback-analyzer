# Project Workspace Shell v1 — SignalForge

## Purpose
Provide the main app shell for the end-to-end workflow inside a project.

## Main layout
- top bar
- project identity area
- workflow step navigation
- main content area
- action row

## Top bar contents
- SignalForge logo
- current project name
- optional project status

## Workflow navigation
- Project
- Inputs
- Insights
- Report
- PRD
- Tasks

## Navigation behavior
- current step highlighted
- previous completed steps accessible
- future steps visible and optionally disabled

## Main content area
This area changes by screen, but the shell remains consistent.

## Action row
Contains:
- primary forward action
- secondary refine/back action

## UX principle
The shell should make the workflow feel linear, stable, and enterprise-ready.
