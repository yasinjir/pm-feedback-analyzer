# Projects Home v1 — SignalForge

## Purpose
Provide a lightweight entry point into the product.

## Goal
Allow users to:
- create a new project
- open an existing project
- resume work from the latest stage

## Main sections
- page header
- create project CTA
- project list
- project card metadata

## Project card fields
- project name
- initiative name
- current workflow stage
- last updated time

## Primary CTA
Create project

## Secondary action
Open project

## UX principle
This screen should feel lightweight, focused, and operational.
