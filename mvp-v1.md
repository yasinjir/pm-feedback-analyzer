# SignalForge MVP v1

## Product
SignalForge

## Current module focus
SignalForge Insights

## MVP goal
Help product teams move from raw feedback to structured insights, PRD drafts, and execution-ready tasks in one workflow.

## Primary user
Product Manager

## Secondary users
- Founder
- Product Analyst
- Product Operations

## Core workflow
1. Create a project
2. Add raw feedback or imported input
3. Generate structured insights
4. Review summary report
5. Generate a PRD draft
6. Generate task drafts
7. Copy or export outputs

## In scope
- project creation
- raw text input
- basic structured input
- CSV import
- insight generation
- report generation
- PRD draft generation
- task draft generation

## Out of scope
- advanced form builder
- real SMS integration
- collaboration
- permissions
- third-party integrations
- automation workflows
- advanced dashboards

## Core outputs
- themes
- pain points
- feature requests
- summary report
- PRD draft
- task list draft

## Success criteria
- A PM can move from raw feedback to structured insight in one session
- A PM can generate a usable PRD draft
- A PM can generate an initial task breakdown
- The workflow feels clear and coherent from input to execution
