# Prototype Build Order v1 — SignalForge

## Objective
Build the first believable product prototype in the smallest number of steps.

## Build strategy
Build the prototype in the same order as the user journey.

---

## Phase 1 — Foundation

### 1. Landing Page
Goal:
Present the SignalForge brand, workflow, and current module clearly.

### 2. Projects Home
Goal:
Show that SignalForge is project-based and workflow-driven.

### 3. Project Workspace Shell
Goal:
Create the reusable app frame for all workflow steps.

Includes:
- top bar
- project title
- workflow navigation
- content area
- action area

---

## Phase 2 — Core workflow proof

### 4. Create Project
Goal:
Show how a PM starts a structured workflow.

### 5. Input Workspace
Goal:
Show how raw feedback enters the system.

### 6. Insights View
Goal:
Demonstrate the key product transformation from input to structured insight.

---

## Phase 3 — Downstream value

### 7. Report View
Goal:
Show stakeholder-friendly synthesis.

### 8. PRD Studio
Goal:
Show how insights become a product document.

### 9. Task Builder
Goal:
Show how the PRD becomes execution-ready work.

---

## Phase 4 — Polish

### Add these after all screens exist
- loading states
- empty states
- success states
- copy feedback
- disabled step behavior
- visual consistency polish

---

## Demo logic rules

### Rule 1
Each screen should feel like a continuation of the previous one.

### Rule 2
Use the same project and content scenario everywhere.

### Rule 3
Do not introduce extra screens that break the workflow.

### Rule 4
Do not add advanced features before the core flow feels complete.

---

## Prototype quality bar
A viewer should be able to understand the product in one walkthrough.

That walkthrough should prove:
- what goes in
- what gets generated
- why the workflow matters
- how the product helps PMs move faster

---

## Recommended order for actual implementation
1. Landing Page
2. Projects Home
3. Project Workspace Shell
4. Create Project
5. Input Workspace
6. Insights View
7. Report View
8. PRD Studio
9. Task Builder
10. Polish and states
